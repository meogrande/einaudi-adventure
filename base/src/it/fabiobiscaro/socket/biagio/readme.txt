Esercitazione in classe del giorno 12/10/2017

Creazione guidata di un applicativo CLI di tipo C/S in linguaggio Java, per l'utilizzo di uno strem buffer I/O al fine di sviluppare una classe Server che si ponga in ascolto su una data porta (parametro passato da linea di comando), in modo da riceve sul proprio socket dei messaggi inviati dai socket di diversi client.
Instaurazione di un protocollo per la chiusura del socket server da remoto.
Funzionamento delle classi Scanner per l'Input stream e PrintWriter per l'Output Stream.
